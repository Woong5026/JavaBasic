기본형식 : 조건식 ? 피연산자1 : 피연산자2

조건식의 연산결과가 true 이면, 결과는 피연산자 1(50)이고, 조건식의 연산결과가 false 이면 결과는 피연산자2(40)

```java

         int b1 = (5>4) ? 50 : 40; 
        //조건식이 true이므로 b1은 50이 된다. 

```

삼항연산자는 if else 와도 같다

```java

            int b2 = 0;
            if(5 > 4){
                b2 = 50;
            }else{
                b2 = 40;
            }

```
